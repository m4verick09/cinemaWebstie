I have used 3 style sheets: Layout.css, ticket.css and Media.css.

Media:

In Media.css, I have added all the code for the styling of images and other media. For example, I added the code for thumbnail borders in Index page and Now Showing
page and I have used this style sheet to align the images relative to the text. This sheet is only linked to the Index page and Now Showing Page.

Ticket:

In this style sheet, I have added the styling for the seat plan and printing of the tickets. This is only linked to the tickets page and print page.
